Git branching

Vad är git branching?

Git skapar ett repository med en branch som andra kan ta del av.

Hur gör man en branch?

Faktum 1. Öppna din terminal 2. Se till att du är i rätt mapp för projektet 3. Skriv "git branch förnamn/namn-på-branchen" 4. Skriv "git checkout" 5. Skriv "git checkout förnamn/namn-på-branchen"

Hur kopplas mina ändringar i branchen till github?

1. Öppna din terminal 2. Skriv "git add förnamn/namnet-på-branchen" 3. Skriv "git commit -m "meddelande på förändringen" 4. Skriv "git push origin förnamn/namnet-på-branchen"
