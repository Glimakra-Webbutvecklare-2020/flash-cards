Vilket element används för att skriva vanlig text i HTML?

p-taggen, som står för "paragraph".

Vad är skillnaden på ett block-element och ett inline-element?

  • a) Ett block-element "blockerar" det som står innanför det så att det inte syns, medan ett inline-element visar det som är innanför det på samma rad.
  • b) Ett block-element skapar alltid en tom linje efter sin slut-tagg, medan ett inline-element fortsätter att skriva på samma rad.
  • c) Alla block-element får automatiskt en svart "box" omkring sig, som liknar ett block, medan inline-element beter sig som vanlig text.
